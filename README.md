# 1. Spring OAuth2

Spring OAuth2 is a project that trains authentication and authorization with delegated access.

# 2. Technologies

- Language: Java (25+)
- Dependency Manager: Maven (3.9.12+)
- Frameworks: Spring Boot (4.0.3+) with Spring Security (7.0.3+)
- Code Versioning: Git (2.53.0+)
- CI: GitHub Actions

# 3. Clone the Repository

- Bash

`git clone https://github.com/jose-techcode/Spring-OAuth2`

# 4. Setting Environment Variable

Create a file called .env in the project root and add your Google client id and Google client secret:

`GOOGLE_CLIENT_ID=`

`GOOGLE_CLIENT_SECRET=`

This file should not be uploaded to GitHub, as it contains sensitive information. Therefore, it should be included in .gitignore.
